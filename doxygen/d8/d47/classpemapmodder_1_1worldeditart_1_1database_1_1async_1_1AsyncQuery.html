abstractprotected

Indicates the expected query result type. Accepted constants:

  • AsyncQuery#RESULT_NIL: no results are expected.
  • AsyncQuery#RESULT_INSERT: the result is expected to contain an integer that specifies the insert ID of the query (if existent)
  • AsyncQuery#RESULT_ASSOC: one row
