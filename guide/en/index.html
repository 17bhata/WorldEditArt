Concept introduction, commands documentation and user tips
NOTE: This guide is written assuming that administrators use the default configuration of WorldEditArt.

WorldEditArt is an innovative plugin. This plugin is not written to "port" the MCPC WorldEdit plugin. This plugin is an entirely original work. Although some commands may follow the convention of WorldEdit to make it easier for understanding, there are a lot of differences that make the plugin easier to use and more adapted for pocket edition users.
In the user guide below, I'm going to explain the features of WorldEditArt one by one.

The most essential component of WorldEditArt is selections.

In conventional world-editing plugins, you run //pos1 at one point and //pos2 at another point (or use wands). This selects the region between the two points. Then you can execute various operations on the selected zone.
However, it has an apparent limitation: no matter how you select the two points, the outcome is still a cuboid. May it be a floor or a wall, it is still a flat cuboid.

WorldEditArt lets you select different shapes of regions, including:

A cuboid is defined by two diagonally opposite points. A cuboid is inscribed between the two points.
It is difficult to explain in words, so here is a diagram that explains it:

In this diagram, AP, BQ and CR are three pairs of diagonal points. To select this cuboid, go to A, run the command //1 (alias for //pos1), then go to B, and run the command //2. By typing //1 or //2 again, you can resize the cuboid by changing one of the points.

Apart from using commands, you can also use wands to select the two points.
This is basically the same as the above method, except that instead of typing a command, you use the defined wand item to define the two diagonal points. By default, clicking an iron hoe on a block sets the block as position 1, while long-clicking it sets the block as position 2.

Another method is to use the //cub s <diagonal length> command. This draws a cuboid by shooting a ray of <diagonal length> blocks long at the direction you are facing. Your current position is position 1, and the other end of the ray is position 2.

Disambiguation: This section is about regular spheres. For the kind of sphere where one of the three radius axes is the longest/shortest, see Ellipsoids.

It is also possible to select regions in spherical shape.

An ellipsoid is a stretched/compressed/distorted sphere. This means, the sphere is no longer of regular radius, but has maximum/minimum radius at the X, Y and Z directions.

This image shows an ellipsoid:

In the image, a and b are horizontal, so they are the X and Z radii respectively. c is vertical, so it is the Y radius.

Syntax to create an ellipsoid selection:

//ell <x radius> <y radius> <z radius>

A right cylinder is a cylinder that either stands vertically or lies horizontally.
A right cylinder is defined by its direction, radius and height.
